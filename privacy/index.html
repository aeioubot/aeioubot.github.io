Privacy policy for the Aeiou Discord bot

Data collection

Aeiou only permanently stores custom reactions that have been set up using the /cradd command. For each custom reaction, the following is stored:

  • Information submitted through the command: trigger, response and matching method
  • ID of the server where the custom reaction is active
  • Time of creation
No personal data is ever collected.

To provide the service, Aeiou scans all messages sent in channels which it has access to for matching custom reactions set up in your server. If you wish to opt out of this behaviour and have all your messages ignored by the bot, send an e-mail to privacy at aeiou.ooo with your Discord user ID.

Data deletion

Custom reactions can be removed using the /cradd command.
To store as little data as possible, user IDs are not collected. It is therefore impossible to find out which reactions you specifically submitted, or to delete those.

Changes to the Privacy Policy

This Privacy Policy may be revised at any time without prior notice.

Questions

If you have any questions, you may contact us by e-mail at privacy at aeiou.ooo.
